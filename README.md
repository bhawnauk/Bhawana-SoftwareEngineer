## 📖 About

This portfolio is where I share who I am as a developer and the projects I've built, the experience I've gained, and how to reach me. It's designed to be fast, responsive, and easy to navigate.

## ✨ Features

- 🗂️ **Projects** — A showcase of my featured work
- 💼 **Experience** — A timeline of my professional background and roles
- 📬 **Contact** — Simple ways to get in touch with me
- 📱 **Responsive Design** — Looks great on desktop, tablet, and mobile
- ⚡ **Fast & Lightweight** — Powered by Vite for instant load times

## 🛠️ Tech Stack

- **Framework:** React
- **Language:** TypeScript
- **Build Tool:** Vite
- **Deployment:** Vercel
- **Styling:** Tailwind CSS

## 📂 Project Structure

The project follows a simple, organized structure inside src/:


components/ — Reusable UI components
pages/ — Page-level components
assets/ — Images, icons, fonts
data.ts — Portfolio data (projects, experience, etc.)
router.tsx — App routing

## 🚦 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone 

# Navigate into the project directory
cd your-dir (Name of your directory)

# Install dependencies
npm install
```

### Development

```bash
# Start the local dev server
npm run dev
```

The app will be available at `http://localhost:5173`.


## ☁️ Deployment

This project is deployed on [Vercel](https://vercel.com). Every push to the `main` branch automatically triggers a new deployment.

To deploy your own version:

1. Push this repository to GitHub
2. Import the project into [Vercel](https://vercel.com/new)
3. Vercel auto-detects the Vite configuration — no extra setup needed
4. Click **Deploy** 🎉
